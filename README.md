# kiosk_project

НА КОД НЕ СМОТРЕТЬ, НАПИСАН СОННЫМ ПАРАЛИТИКОМ ПОД БОЛШИМ КОЛИЧЕСТВОМ ПИВА!!!

Проект бота для заказа товаров из ларька для самовывоза

Python 3.13.2

## Пользователи

- id - ID
- username - имя пользователя
- telegram id - телеграм ID
- blocker - заблокирован ли пользователь

## Точки

- id - ID
- address - адрес
- user_id - пользователь ID

## Заказы

- id - ID
- user_id - пользователь ID
- kiosk_id - точка ID
- status - статус
- paid - оплачен
- cost - стоимость

## Товары которые могут быть

- id - ID
- name - наименование
- units_meas - единицы измерения

## Товары в наличии на точке

- id - ID
- product_id - товар ID
- kiosk_id - точка ID
- count - количество в наличии
- price - цена

## Товары в заказе

- id - ID
- order_id - заказ ID
- product_id - товар ID
- count - количество
